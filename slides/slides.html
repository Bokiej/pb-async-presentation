How to use finally() and loader?

Loader / Spinner is a special component that creates animation that shows User that data is downloading and he have to wait until it finish. finally() method is a perfect place to stop loader / spinner because error or success occoured.

finally() method

Loader or lightning-spinner should be added to lwc HTML with if:true argument and reference to value in js. In xe it is isloading value;

finally() method

Value isLoading controls render of our loader component. At the begining is true and after Promise is finished it changes to false what removes loader from DOM.
