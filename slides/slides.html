Does the location of the Promise methods matters?

Promise has three different methods then(), catch(), finally().

Promise composition

Every of then(), catch(), finally() methods will return an equivalent Promise object. This allow to chain calls to another promise method. console.logs show order of invoking methods in Promise chain. Logs in browser console will return values in below order: 

                    
                        1. setBulbColor
                        2. setBulbTime
                        3. setBulbCounter
                        4. showBulbNumber
                        finally method
