How it works

Presentation is created no only to talk about slides but also for exloring and lerining it by yourself. Every section is separated git branch where are only necessary LWC components discussed at the time.

After clicking this button:

git branch will be automatically changed to next section and all files will be switched to only necessary one.

After installation everything like described in README.md in main directory you will have access to two node servers.

You run presentatnion by writing npm start in system console being in main project directory what will open in browser localhost:8000.

You run Local Development Server by writing npm run serve in system console being in main project directory and open in browser localhost:3333.

There are two servers, one for presentation and one for LWC local server.

node.js server
npm strat
localhost:8000
Salesforce Local Development Server
npm run serve
localhost:3333

Every presentation section have its own structure you can watch by clicking Escape button.

While I will talking about presentation I will go only right (click right navigation arrow or right button in keyboard) [red square in screen].

But for your own learning explore slides by going down (click down arrow navigation or down button in keyboard) to lear more [orange square in screen].
